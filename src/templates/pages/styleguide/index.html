Introduction

Welcome to the isobar boilerplate project.

Forking this repo gets you up and running with a new project quickly and lets you leverage some neat tools that we hope you find useful.

The directory structure is opinionated, and makes some assumptions for you on how things should be organised, this simplified how most projects usualy run and is quite flexible. If you would like to set things up differently, free to change these as you see fit, but please make sure that these are reflected and updated in the styleguide too.

This project comes bundled with a few base atoms and some examples of how you create more complex components, where their HTML/SCSS/JS live and how you can incorporate these in the styleguide.

Atoms

These are the base building blocks of most HTML pages. They are simple single tag items, possibly two as in the case of a form input with a label, but nothing more complex.

Things more abstract such as a colour and a grid entry are also classified here as atomic.

Herea are the pre-made ones in this project.

  • Grid
  • Colours
  • Typography
  • Buttons
  • Forms

Molecules

The next level in the complexity chain, comes molecules. These are for items composed of two or more atomic tags, but that are still simple in conception and have no JS interactivity. Examples of these could be a simple navigation, a specific form, a grid of image links.
